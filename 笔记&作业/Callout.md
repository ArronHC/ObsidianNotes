## 介绍
Callout 是用于在文档中创建**醒目、突出**的标签块所设计的语法。
## 语法
```Markdown
> [!关键字]
> 语句
```
- **关键字**：
	- `NOTE`：注意
	- `INFO`：信息
	- `TODO`：代办
	- `TIP / HINT`：提示/技巧
	- `SUCCESS / CHECK / DONE`：成功 / 完成
	- `QUESTION / HELP / FAQ`：问题
	- `WARNING / CAUTION`: 警告 / 注意
	- `DANGER / ERROR`: 危险 / 错误
	- `EXAMPLE` : 示例
	- `QUOTE / CITE`：引用
>[!note] 注意事项
> 注意啦！

> [!info]
> 这是一条信息

> [!warning]
> 警告！不要阅读这条消息

> [!example]
> ```python
> print("Hello World!")
> ```
